# ClearPath Environmental — Business Plan Site

Professional one-page business plan for an environmental cleanup venture based in Oregon City / Clackamas County.

## Tech Stack
- React 19 + Vite
- Netlify hosting with SPA redirects
- Google Fonts (DM Serif Display + Source Sans 3)

## Netlify Site
- **Site ID:** `0202e798-6027-4492-85b6-06580108bd21`
- **URL:** https://clearpath-environmental.netlify.app

## Local Dev
```bash
npm install
npm run dev
```

## Customization
Edit the config at top of `src/App.jsx` to change name/tagline/region.

## Connecting to Netlify
1. Push this repo to GitHub
2. Link at https://app.netlify.com/projects/clearpath-environmental
3. Build command: `npm run build` | Publish dir: `dist`
